## AC1 — Snap-to-grid quantizes clip placement

Snap is **on by default at 1-beat resolution** (magnet toggle pressed in the toolbar), and lanes draw grid lines from the same step the snapping uses. The AI Song card was drag-dropped onto an AI track at **x = 137px (1.37s — deliberately off-grid)**; the clip landed at 150px = 1.5s, the nearest beat line.

```bash
agent-browser eval "
const lane = document.querySelector(String.raw\`[role=region][aria-label=\"Track 1 timeline\"]\`);
const block = lane.querySelector(\"[data-testid=clip-block]\");
JSON.stringify({snapPressed: document.querySelector(String.raw\`[aria-label=\"Snap to grid\"]\`).getAttribute(\"aria-pressed\"), gridLines: lane.style.backgroundImage.slice(0, 46), droppedAtPx: 137, clipLandedAt: block.style.left})"
```

```output
"{\"snapPressed\":\"true\",\"gridLines\":\"repeating-linear-gradient(to right, var(--bord\",\"droppedAtPx\":137,\"clipLandedAt\":\"150px\"}"
```

Switching the resolution dropdown to **1 bar** (2s = 200px) and dragging the placed clip to x = 333px (3.33s): it snaps to the nearest bar line at 4s = 400px.

```bash
agent-browser eval "
const lane = document.querySelector(String.raw\`[role=region][aria-label=\"Track 1 timeline\"]\`);
const block = lane.querySelector(\"[data-testid=clip-block]\");
const br = block.getBoundingClientRect(); const lr = lane.getBoundingClientRect();
const dt = new DataTransfer();
block.dispatchEvent(new DragEvent(\"dragstart\", {bubbles: true, dataTransfer: dt, clientX: br.left}));
const x = lr.left + 333;
lane.dispatchEvent(new DragEvent(\"dragover\", {bubbles: true, cancelable: true, dataTransfer: dt, clientX: x}));
lane.dispatchEvent(new DragEvent(\"drop\", {bubbles: true, cancelable: true, dataTransfer: dt, clientX: x}));
\"dropped at 333px\"" && sleep 0.5 && agent-browser eval "
const b = document.querySelector(\"[data-testid=clip-block]\");
JSON.stringify({resolution: document.querySelector(String.raw\`[aria-label=\"Snap resolution\"]\`).textContent, movedTo: b.style.left})"
```

```output
"dropped at 333px"
"{\"resolution\":\"1 bar\",\"movedTo\":\"400px\"}"
```

## AC2 — Loop region repeats playback

The transport gained a **loop toggle** (repeat icon). Enabling it shows the default loop band over the ruler (0s–8s = 4 bars at 120 BPM) with draggable start/end bracket handles — real ARIA sliders with keyboard support. The end handle was dragged from 8s to **4s (400px)**.

```bash
agent-browser click "button[aria-label=\"Toggle loop\"]" && sleep 0.5 && agent-browser eval "
const region = document.querySelector(\"[data-testid=loop-region]\");
const handle = document.querySelector(String.raw\`[aria-label=\"Loop end handle\"]\`);
const area = document.querySelector(\"[data-testid=ruler-area]\");
const ar = area.getBoundingClientRect();
handle.dispatchEvent(new PointerEvent(\"pointerdown\", {bubbles: true, pointerId: 1, clientX: ar.left + 800}));
handle.dispatchEvent(new PointerEvent(\"pointermove\", {bubbles: true, pointerId: 1, clientX: ar.left + 397}));
handle.dispatchEvent(new PointerEvent(\"pointerup\", {bubbles: true, pointerId: 1}));
\"end handle dragged 800px -> 397px\"" && sleep 0.5 && agent-browser eval "
const region = document.querySelector(\"[data-testid=loop-region]\");
const handle = document.querySelector(String.raw\`[aria-label=\"Loop end handle\"]\`);
JSON.stringify({loopPressed: document.querySelector(String.raw\`[aria-label=\"Toggle loop\"]\`).getAttribute(\"aria-pressed\"), bandLeft: region.style.left, bandWidth: region.style.width, endHandleValue: handle.getAttribute(\"aria-valuenow\")})"
```

```output
[32m✓[0m Done
"end handle dragged 800px -> 397px"
"{\"loopPressed\":\"true\",\"bandLeft\":\"0px\",\"bandWidth\":\"400px\",\"endHandleValue\":\"4\"}"
```

With the loop active (0s–4s), pressing **Play** and sampling the playhead (left = 160px strip + sec × 100px) shows it advancing, then wrapping back inside the loop instead of passing 4s = 560px. Audio reschedules from the loop start on every wrap via the same seek path a manual ruler-click uses.

```bash
agent-browser click "button[aria-label=\"Return to start\"] + button" && sleep 2 && agent-browser eval "\"t≈2s playhead at \" + document.querySelector(String.raw\`[data-testid=\"playhead\"]\`).style.left" && sleep 3 && agent-browser eval "\"t≈5s playhead at \" + document.querySelector(String.raw\`[data-testid=\"playhead\"]\`).style.left" && sleep 2 && agent-browser eval "\"t≈7s playhead at \" + document.querySelector(String.raw\`[data-testid=\"playhead\"]\`).style.left"
```

```output
[32m✓[0m Done
"t≈2s playhead at 367.819px"
"t≈5s playhead at 266.812px"
"t≈7s playhead at 471.147px"
```

All samples stay inside the 0–4s loop window (playhead x between 160px and 560px): playback crossed the loop end twice and wrapped back both times. Toggling loop **off** mid-play lets the same run sail straight past the old boundary:

```bash
agent-browser click "button[aria-label=\"Toggle loop\"]" && sleep 5 && agent-browser eval "

```

```output

[32m✓[0m Done
```

## AC3 + AC4 — Named markers: add, render, rename, move, delete

**Add**: double-clicking the ruler strip at x = 612px (6.12s, snap back on at 1 beat) creates a flag labeled "Marker 1" snapped to 6s = 600px, rendered on the marker strip above the ruler.

```bash
agent-browser eval "
const area = document.querySelector(\"[data-testid=ruler-area]\");
const ar = area.getBoundingClientRect();
area.dispatchEvent(new MouseEvent(\"dblclick\", {bubbles: true, clientX: ar.left + 612}));
\"double-clicked ruler at 612px\"" && sleep 0.5 && agent-browser eval "
const flags = [...document.querySelectorAll(String.raw\`[aria-label^=\"Marker:\"]\`)];
JSON.stringify(flags.map(f => ({label: f.getAttribute(\"aria-label\"), left: f.style.left})))"
```

```output
"double-clicked ruler at 612px"
"[{\"label\":\"Marker: Marker 1\",\"left\":\"600px\"}]"
```

**Rename**: clicking the flag opens its editor popover; typing "Chorus" and hitting Rename updates the flag label in place.

```bash
agent-browser click "[aria-label=\"Marker: Marker 1\"]" && sleep 0.7 && agent-browser fill "input[aria-label=\"Marker label\"]" "Chorus" && agent-browser click "button[type=submit]" && sleep 0.5 && agent-browser eval "
const flags = [...document.querySelectorAll(String.raw\`[aria-label^=\"Marker:\"]\`)];
JSON.stringify(flags.map(f => ({label: f.getAttribute(\"aria-label\"), left: f.style.left})))"
```

## Result

All four acceptance criteria verified live: snap quantized off-grid drops/moves at both 1-beat and 1-bar resolutions (and stopped when toggled off), loop playback wrapped twice inside 0–4s then played through once disabled, and markers were added, renamed, moved (snapped), and deleted on the ruler. Loop handles are keyboard-adjustable ARIA sliders (arrow keys nudge by one snap step).
